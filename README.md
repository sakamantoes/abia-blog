# Abia State Digital Blog & Information Platform

A full-stack blog platform for Abia State built with React, Node.js, Express, MySQL, and Cloudinary.

## Features

- User authentication (register/login)
- Role-based access (Admin & Regular users)
- Blog post management (Admin only)
- Comment system with edit/delete functionality
- Like/unlike posts
- Image upload via Cloudinary
- Responsive design with Tailwind CSS
- Homepage with sections: Hero, About, Services, Tour Abia, Map, Contact

## Tech Stack

**Frontend:** React.js, Tailwind CSS, React Router, Axios, React Quill

**Backend:** Node.js, Express.js, Sequelize ORM, MySQL, JWT, bcrypt

**Image Storage:** Cloudinary
